Welcome to NeuralGolf!

Neural golfing applies the ideas beind code golfing to neural networks. This website shows the leaderboard for the most compressed models on various different datasets. If you want to compete, check out the rules for your dataset of choice and start training! If you have a score that lands on the leaderboard them submit your model here.

If you would like to propose a new category then submit one here.
